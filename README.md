# School Alumni Social Network (Demo)

Минимално уеб приложение с Flask за училищна социална мрежа.

## Функционалности
- Регистрация с роли: ученик (учащ/завършил), учител, зам.-директор, директор, системен администратор.
- Проверка на ученическите регистрации спрямо вътрешна база `school_directory`.
- Статуси за регистрация: `approved/pending/rejected`.
- Профили с редакция, работно място, био и поверителност.
- Публикации със снимки.
- Сторита (валидни 24 часа).
- Преглед и одобрение/отказ на регистрации (директор/системен админ).
- Responsive интерфейс за мобилни и desktop устройства.

## Стартиране
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Отвори: http://127.0.0.1:5000
