# School Alumni Social Network

Пълнофункционално уеб приложение за училищна социална мрежа (ученици, завършили, учители и ръководство).

## Какво има
- Регистрация с проверка за три имена + клас + година на завършване.
- Валидация на ученици спрямо училищен регистър (`school_directory`).
- Потвърждение на регистрации от директор/системен администратор.
- Роли и права: ученик, учител, зам.-директор, директор, системен админ.
- Профили, био, работно място, аватар, поверителни профили.
- Follow заявки за поверителни профили.
- Публикации със снимки.
- Сторита (24 часа).
- Админ панел за регистрации и училищен регистър.
- Готово за GitHub (CI workflow).

## Стартиране локално
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Отваряш: http://127.0.0.1:5000
